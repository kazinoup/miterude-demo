# ミテルデ - demo

IoT センサー（温湿度）の状態監視・運用記録・レポート出力をブラウザだけで完結するデモアプリ。
ダッシュボード、センサー一覧、ゲートウェイ管理、確認記録、月報／週報の PDF 出力までを 1 つの SPA で扱います。サーバーや DB は不要で、データはブラウザの `localStorage` に保存されます。

🌐 **公開デモ:** https://miterude-demo.vercel.app/

## 主な機能

- **ダッシュボード** — タイル／フロアマップ／時系列チャート／逸脱ピックアップを組み合わせたウィジェット構成
- **センサー一覧** — フィルタ・タグ・グループ・ユーザー定義区分（アイコン付き）・列の表示設定
- **確認記録** — ダッシュボードのスナップショット＋逸脱コメントの記録、承認フロー
- **月報／週報** — A4 PDF 出力（温度・湿度の逸脱判定、欠損補完、サマリ＋グラフ）
- **CSV インポート** — IoT センサーの CSV をブラウザで読み込んでデバイス／ゲートウェイを自動生成

## 前提

- CSV 列: `時間`, `温度`, `湿度`, `バッテリー`（バッテリーは任意）
- ファイル名（拡張子除く）がデバイス名になります（例: `CK27.csv` → `CK27`）
- 温度・湿度は `" -21.6 ℃"`, `"72 %"` のように単位付きでも可（先頭の数値を抽出）

## 使い方

```bash
npm install
npm run dev
```

開発サーバーは **http://localhost:3100/** で起動します（`vite.config.ts` の `server.port`）。

1. CSV を複数選択してアップロード
2. 対象デバイス・対象月、温度・湿度の逸脱閾値、欠損表示を設定
3. 「レポートを表示」→ スクロールでプレビュー
4. 「PDF 用に印刷」→ ブラウザの印刷から「PDF に保存」、用紙は A4 縦推奨

## 仕様メモ

- 同一 30 分枠に複数行がある場合は平均値でセルに表示
- 欠損は空欄または `-`（設定で切替）
- 逸脱セルは赤表示。逸脱回数はスロット（30 分枠）単位でカウント
- 計測回数・平均・最大・最小は、その月の生データ行に基づく
